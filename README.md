## Sample Code

```dart
BubbleRadioButton<int>(
    activeColor: Colors.Green,
    inactiveColor: Colors.white,
    activeTextColor: Colors.white,
    inactiveTextColor: Colors.bluegrey,
    fontSize: 18,
    textColor: Colors.white,
    value: "A",
    groupValue: A,
    leading: 'A',
    onChanged: (value) {
    setState(() {
       _value = value!
    });
  },
),
```
